fb.js Demo Pages

CloudPages running outside JXBrowser using DemoAdapter with realistic inline data. All pages use FB.configure({ environment: 'demo', demoData: {...} }).

How Demo Mode Works

When environment: 'demo' is set, fb.js uses the DemoAdapter which:

  • Queries — matched against keys in demoData.queries (exact match, substring, or <script> tag ID)
  • User/Context info — returned from demoData.user and demoData.context
  • Plugin data — stored in memory + localStorage
  • UI methods — update DOM elements (#fb-status, #fb-progress)
  • Sync methods — throw PlatformError (use Async variants)
